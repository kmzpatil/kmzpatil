## Featured Projects

### Gloser AI - Pharmaceutical Intelligence Platform
A multi-agent AI system that processes natural language queries to analyze patents, clinical trials, and trade data. Features real-time markdown rendering and chart visualization.
> **Tech Stack:** `Next.js`, `LangGraph`, `Ollama`, `Python`, `Flask`

### GMUN 2026 - Official Event Platform
The official digital interface for the Global Model United Nations at IIT KGP. Handled over **300+** delegate registrations seamlessly with a mobile-first responsive design.
> **Tech Stack:** `React.js`, `Node.js`, `CSS3`

### Bhavya Haulage - Logistics Marketing Site
A high-performance landing page for a logistics firm featuring sticky navbars, lazy loading, and 60fps animations for optimal UX.
> **Tech Stack:** `React`, `Vite`, `Tailwind CSS`, `Framer Motion`

### Cancer Detection - Predictive ML Model
Secured **1st Runner-up** in the KDAG Machine Learning Hackathon. Developed a predictive model to identify cancer markers with high accuracy using patient data.
> **Tech Stack:** `Python`, `Scikit-Learn`, `Pandas`, `NumPy`
